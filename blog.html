Important FAQ

What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a way for web browsers to organize HTML as a tree of elements. It lets JavaScript change or update the content, structure, and style of a webpage. For example, you can use the DOM to change text, hide elements, or add new content to a page dynamically.

How do you select an element from the DOM ?

In JavaScript, selecting an element from the DOM refers to the process of retrieving a specific HTML element from the document using methods like `getElementById`, `getElementsByClassName`, `getElementsByTagName`, `querySelector`, and `querySelectorAll`. This allows you to interact with and manipulate the selected element(s) programmatically.

What is event delegation in the context of the DOM, and why is it useful?

**Event delegation** is a technique in JavaScript where a single event listener is attached to a parent element to manage events for multiple child elements. It leverages event bubbling to handle events efficiently, improving performance, simplifying code, and automatically managing events for dynamically added elements.

How do you maniplate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using `setAttribute`, `getAttribute`, and `removeAttribute`, and change its styles using `element.style.propertyName`.
